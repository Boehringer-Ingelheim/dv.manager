# A simple module that counts the number of rows for all datasets loaded in the application

This simple module is used for demonstration purposes in documentation,
testing and developing

## Usage

``` r
multi_simple_UI(id)

multi_simple_server(id, dataset)

mod_multi_simple(module_id)
```

## Arguments

- id:

  shiny id

- dataset:

  input dataset

- module_id:

  shiny module ID

## Functions

- `multi_simple_UI()`: Module UI

- `multi_simple_server()`: Module server

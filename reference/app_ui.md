# The application User-Interface using an HTML template

The application User-Interface using an HTML template

## Usage

``` r
app_ui(request_id)
```

## Arguments

- request_id:

  This parameter can be either an string id if the app_ui is being used
  as a module or a request if the app_ui is being used as an standalone
  application then it becomes a Internal parameter for `{shiny}` DO NOT
  REMOVE.

# A simple module that counts the number of rows

This simple module is used for demonstration purposes in documentation

## Usage

``` r
simple_UI(id)

simple_server(id, dataset)

mod_simple(dataset, from, module_id)
```

## Arguments

- id:

  shiny id

- dataset:

  name of the dataset

- from:

  name of the dataset_list

- module_id:

  shiny module ID

## Functions

- `simple_UI()`: Module UI

- `simple_server()`: Module server
